Работает только со следующими типами:text,checkbox,radio
Также в силу того что в выбранном файле могут быть выбраны не все checkbox,программа может отметить не все верные checkbox
Данный проект является программой,для переноса данных из html 
файла на сайт Мосполитеха.
Он работает следующим образом,вы входите на сайт через gui,далее
выбираете необходимый тест и курс и файл с ответами на нужный
вам тест(html).Затем программа переносит данные с файла с ответами 
на выбранный тест.
Для запуска достаточно скачать необходимые библиотеки и запустить файл new_gui_version.py
new_gui_version.py-это сам GUI приложения
programm.py-файл,содержащий все запросы и их обработку на сайт Мосполитеха
test.py-не особо нужный файл,так как тесты не очень удобно проводить,из-за данных на сайте,которые меняются независимо от программы
responce_files-это файлы html,которые представляют собой ответы Мосполитеха на запросы(на вход,выбор курса,открытие попытки и так далее)
тесты-сами тесты,которые можно использовать для программы
Как программа находит и вставляет необходимые ответы:
Для начала находятся ответы в выбранном файле,для этого выбираются все теги с определённым классом,в которых содержаться вопросы,далее проходимся
по каждому тегу,определяя его тип(radio,checkbox,text) и в зависимости от типа обрабатываем его с помощью необходимой функции,которая возвращает
ответ(ы) и вопрос с данного тег,затем складываем ответы и вопросы в словарь,потом проходимся по каждой странице теста и собираем данные,которые
необходимо заполнить и в зависимости от заполняемой формы вызываем функцию,которая пытается найти ответы на необходимый вопрос и подбирает данные
необходимы для post запроса.
